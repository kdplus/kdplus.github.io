  2. Command Version

配置

GUI版本add一個之後按照要求填寫即可。
下面是命令版本的配置

  1. 首先你需要創建一個config來存連接ss的信息 裏面有這些如下字段 sslocal -c config.json sudo vim ss_start.sh
    sslocal -c /YOUR_PATH/config.json sudo vim /etc/rc.local sudo sh /YOUR_PATH/ss_start.sh
